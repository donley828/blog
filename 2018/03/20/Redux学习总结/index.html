这个函数用来管理各个分离的小 Reducer

3.Store

Stroe 就是整个应用存放数据的地方，一个应用只有一个 store。

  • store 可以维持应用的 state
  • 提供 getState() 方法获取 state
  • 通过 dispatch(action) 更新 state
  • 通过 subscribe(listener) 注册监听器
  • replaceReducer用来替换 store 当前用来处理 state 的 reducer
